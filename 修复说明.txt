修复时间：2025年11月23日

【已修复的问题】

1. tkinter Treeview 错误
   ────────────────────────────────────────────
   错误信息：
     _tkinter.TclError: unknown option "-text"
   
   原因：
     Treeview的column()方法不支持text参数
   
   修复：
     移除了column()方法中的text参数
     text参数只能在heading()方法中使用
   
   影响文件：
     sysinternals_gui.py (第111行)

2. 批处理文件编码问题
   ────────────────────────────────────────────
   错误信息：
     '�具集' is not recognized as an internal or external command
   
   原因：
     批处理文件中的中文字符在某些系统环境下显示乱码
     Windows批处理对UTF-8编码支持不完善
   
   修复：
     将批处理文件中的所有中文改为英文
     移除了特殊Unicode字符（如━等）
   
   影响文件：
     启动工具.bat
     以管理员启动.bat

【测试验证】

✓ Treeview组件创建成功
✓ column方法参数正确
✓ 批处理文件可以正常运行
✓ 管理员权限请求正常

【使用方法】

现在可以正常使用以下任一方式启动程序：

方式1：双击 启动工具.bat
方式2：双击 以管理员启动.bat（推荐）
方式3：命令行运行 python sysinternals_gui.py

【注意事项】

• 如需访问系统进程，请使用"以管理员启动.bat"
• Python版本要求：3.6 或更高
• 确保Python已正确安装且添加到PATH环境变量
